# BedReset
A Minecraft Plugin that Resets the phantom insomnia counters after you clicked the bed, despite whether you slept or not

## Features
- Reset insomnia during daytime
- Optional once-per-night restriction
- Cooldown system
- Suppresses vanilla sleep errors
- Configurable behavior

## Installation
1. Download the latest release
2. Drop into your plugins folder
3. Restart server
4. Edit config.yml if needed

## Configuration
Edit config.yml after first run.
